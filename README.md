## Install Apollo

```bash
npm install vue-apollo graphql@15.8.0 apollo-client apollo-link apollo-link-http apollo-cache-inmemory graphql-tag
```

## Install Codegen

```bash
npm install @graphql-codegen/cli -D
npm install @graphql-codegen/typescript -D
```
